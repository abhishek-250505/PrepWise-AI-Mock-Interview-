# PrepWise AI

PrepWise AI is a simple tool I built to practice interviews in a more realistic way.

Instead of solving random questions, it takes your resume and generates questions based on your actual skills and projects. Then it evaluates your answers and gives feedback — kind of like a real interviewer.

---

## 🚀 What it does

- Upload your resume (PDF)
- Extracts skills and projects
- Generates interview questions (HR + behavioral)
- Lets you answer in a mock interview format
- Gives feedback with scores and suggestions

---

## 💡 Why I built this

Most platforms give generic questions that don’t match your background.  
I wanted something that feels more personal and closer to a real interview.

---

## 🛠 Tech Stack

- Frontend: React + Tailwind
- Backend: Node.js + Express
- AI: OpenAI API
- Resume Parsing: pdf-parse

---

## ⚙️ How it works

1. User uploads resume
2. Backend extracts text
3. AI generates questions based on resume
4. User answers questions
5. AI evaluates answers and returns feedback
